## 💼 USER JOURNEY

### Journey 1: Single Course Purchase
1. Browse courses → Select course
2. Click "Enroll Now" button
3. Directed to `register.html`
4. Register for single course
5. Pay per-course price

### Journey 2: Subscription (All-Access)
1. Browse courses → Select course
2. Click "View All-Access Plans" or "Get All-Access Pass"
3. Directed to `plans.html`
4. Compare subscription options
5. Choose Free/Monthly/Annual plan
6. Get access to ALL courses
