Course Modules

Module 1: Cloud Computing Fundamentals

4 Hours
  • Understanding cloud computing benefits and models
  • IaaS vs PaaS vs SaaS comparison
  • Public, private, and hybrid cloud scenarios
  • Capital vs operational expenditure (CapEx vs OpEx)
  • Cloud service scalability and elasticity
  • Lab: Compare cloud deployment models

Module 2: Azure Architecture & Services

5 Hours
  • Azure regions and availability zones
  • Resource groups and subscriptions
  • Azure Portal, CLI, and PowerShell interfaces
  • Azure Resource Manager (ARM) fundamentals
  • Management groups and organization hierarchy
  • Lab: Navigate Azure Portal and create resource groups

Module 3: Azure Compute Services

4 Hours
  • Virtual Machines deployment and management
  • Azure App Service and web applications
  • Container services (ACI and AKS basics)
  • Azure Functions serverless computing
  • VM scaling and availability sets
  • Lab: Deploy and connect to Windows VM

Module 4: Storage & Databases

3 Hours
  • Blob storage, file shares, and disk storage
  • Storage account types and replication
  • Azure SQL Database fundamentals
  • Cosmos DB and NoSQL concepts
  • Database backup and high availability
  • Lab: Create storage account and upload files

Module 5: Azure Networking

3 Hours
  • Virtual Networks (VNets) and subnets
  • Network Security Groups (NSGs) and rules
  • VPN Gateway and ExpressRoute basics
  • Azure Load Balancer and Traffic Manager
  • Public and private IP addressing
  • Lab: Configure VNet and test connectivity

Module 6: Security & Identity

4 Hours
  • Azure Active Directory fundamentals
  • Multi-Factor Authentication (MFA)
  • Role-Based Access Control (RBAC)
  • Azure Security Center and Defender
  • Key Vault for secrets management
  • Lab: Create Azure AD users and assign roles

Module 7: Governance & Compliance

2 Hours
  • Azure Policy and compliance enforcement
  • Resource locks and tagging strategies
  • Azure Blueprints for standardization
  • Compliance certifications and standards
  • Privacy and data protection regulations
  • Lab: Implement Azure Policy and resource tags

Module 8: Cost Management & SLAs

2 Hours
  • Azure pricing calculator and TCO calculator
  • Cost Management tools and budgets
  • Azure Advisor recommendations
  • Service Level Agreements (SLAs)
  • Resource optimization strategies
  • Lab: Analyze costs and set budget alerts

Hands-On Labs & Projects

Lab 1: Deploy Azure Virtual Machine

Create Windows/Linux VMs, configure networking, and connect via RDP/SSH in live Azure environment.

Lab 2: Configure Storage Solutions

Set up storage accounts, upload blobs, create file shares, and implement storage replication strategies.

Lab 3: Build Virtual Network

Design VNet architecture, configure subnets, implement NSGs, and test connectivity between resources.

Lab 4: Azure AD & RBAC

Create Azure AD users/groups, assign RBAC roles, enable MFA, and manage access permissions.

Lab 5: Deploy Web Application

Deploy App Service, configure custom domains, implement SSL certificates, and monitor performance.

Lab 6: Implement Security Controls

Configure Azure Security Center, enable Defender, create Key Vault, and implement security policies.

Lab 7: Monitor & Optimize Costs

Use Cost Management tools, analyze spending, create budgets, and implement optimization recommendations.

Project: Build Complete Azure Solution

Design and deploy multi-tier application with VMs, storage, networking, security, and monitoring.
